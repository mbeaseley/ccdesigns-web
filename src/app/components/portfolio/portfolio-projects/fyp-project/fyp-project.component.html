Final Year Project

My final year project at Brunel University was title "Using Smartphone technology to simulate human behaviour within an indoor environment".

Summary of the project is as followed:

  • Created an Android Application which allowed a user to navigate through an indoor environment.
  • Project included the use of NFC, Beacons, Wi-Fi, and Java.
  • The model used was the Waterfall Model due to the scope of the project but Android development was completed using a similar process that the Spiral Model presented.
  • A literature review was conducted as well as a survey data was researched to underline the requirements of the application.
  • Created Low and High fidelity prototypes of the User Interface, whereas Class, Sequential and Use case diagram was created to developed how and what variables and processes are needed.
  • The development of the application was conducted on Android Studio alongside an external Software Development Kit from Indooratlas.
  • The application was tested using requirement verification as well as performance testing through GPU and CPU monitoring.
  • The application was evaluated against major application on battery usage, accuracy, and loading times.
