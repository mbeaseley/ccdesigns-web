Personal Website Project

This project is the first actual project live in production to be view. Purpose of my personal website is to act like a CV and a promotion for all further projects created by myself.

Summary of the project is as followed:

  • Website is created in Angular 6 using Bootstrap 4.
  • Content is supplied by Contentful (currently just assets).
  • Azure and Azure DevOps is the cloud computering platform used to put website into Production.
  • Continous Integration is set up based on any commit made in the master branch on git.
  • Azure DevOps, as well as the codebase, uses yarn to install dependencies and run a build for production.
  • Created the project in github following gitflow principles.
  • Website was designed with paper and digitaled via Axure RP.
